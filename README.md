# HomeLab

Este repositorio contiene información sobre la configuración de mi _HomeLab_ con la idea de poder compartirlo con amigos y compañeros que necesiten ejecutar servicios similares.

La idea es que esta información sea lo más neutra posible pero, al mismo tiempo, que pueda servir para recuperar desde cero mi entorno en caso de desastre.

Por este motivo, la información será muy esquemática o "paso a paso" simples que se puedan seguir fácilmente si se tiene un mínimo conocimiento del sistema operativo, tecnología o servicio que se describe.

Si tengo información más detallada en algún artículo de [mi blog personal](https://www.manelrodero.com/blog/) se hará referencia a la misma para poder ampliar o completar la información.

## Servidores

Mi pequeño laboratorio en casa está formado por tres "**servidores**" en los que se ejecutan múltiples [aplicaciones](./Apps/README.md) y [scripts](./Scripts/README.md):

* **Raspberry Pi 4** ejecutando **Raspberry Pi OS Lite**
* **Dell OptiPlex 7050 Micro** ejecutando **Proxmox Virtual Environment**
* **PC clónico** con formato torre ejecutando **Unraid**

Y dos ordenadores "**cliente**" desde los que trabajo habitualmente:

* **PC clónico** con formato torre ejecutando **Windows 11**
* **Dell Precision 7530** ejecutando **Windows Server 2022**
